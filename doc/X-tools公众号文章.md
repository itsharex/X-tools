### 五、3步上手X-tools，就是这么简单

#### 第1步：安装软件
```bash
# 1. 克隆仓库（或直接下载安装包）
git clone https://github.com/yan/X-tools.git

# 2. 安装依赖
npm install

# 3. 启动软件
npm start
```

#### 第2步：添加你的资料
- 点击「添加文件夹」按钮
- 选择你存放学习资料的目录
- X-tools会自动扫描并显示所有文件

#### 第3步：开始使用
- 点击左侧文件树浏览文件
- 使用顶部搜索框查找内容
- 在右侧预览区阅读/观看资料

**是不是超级简单？1分钟就能学会！**
